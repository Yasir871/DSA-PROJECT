# Student Management System (Java)

## Project Overview
This project is a Java-based console application developed using IntelliJ IDEA.
It demonstrates the use of Object-Oriented Programming concepts along with
Data Structures and Algorithms.

The system stores student records and allows sorting and searching
of students based on their ID.

## Features
- Stores student information (ID, Name, Marks)
- Sorts students using Insertion Sort
- Searches students using Binary Search
- Demonstrates inheritance and polymorphism

## Technologies Used
- Java
- IntelliJ IDEA

## Algorithms Used
- Insertion Sort
- Binary Search

## How to Run the Project
1. Open the project in IntelliJ IDEA
2. Open the Main.java file
3. Click the Run button
4. View output in the console
